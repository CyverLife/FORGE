# Reglas de Diseño: Sistema de Rangos de Conciencia
**ESTADO: INMUTABLE**

Este documento define las reglas estrictas para la representación visual de los Rangos de Conciencia. Estas reglas NO deben ser modificadas sin orden explícita del Usuario.

## 1. Bronce - "Impulso Driven" (0-49 pts)
- **Concepto**: La materia prima, el estado no refinado.
- **Estética**: Piedra bruta, textura áspera, bordes irregulares, pesado.
- **Paleta**: 
  - Base: `forge-black` (#0B0B0C)
  - Acento: Cobre apagado / Tierra (#8D6E63)
  - Acabado: Mate, sin brillo.

## 2. Plata - "Ego Driven" (50-74 pts)
- **Concepto**: La herramienta afilada, la autodisciplina fría.
- **Estética**: Metal pulido pero frío, industrial, bordes afilados y precisos.
- **Paleta**:
  - Base: Gris Metálico (#9CA3AF)
  - Acento: `forge-steel` (#2A2A2E)
  - Acabado: Satinado, reflejos fríos (Azulados sutiles).

## 3. Oro - "Purpose Driven" (75-94 pts)
- **Concepto**: La maestría, el valor forjado.
- **Estética**: Metal brillante, geometría sagrada/definida, perfección.
- **Paleta**:
  - Base: Dorado Cálido (#F59E0B)
  - Acento: `forge-red` (#C21F1F) (sutil, en detalles)
  - Acabado: Brillante, radiante.

## 4. Infinito - "Transcendent" (95-100 pts)
- **Concepto**: La energía pura contenida, más allá de la materia.
- **Estética**: Obsidiana pulida rompiéndose por luz interior.
- **Paleta**:
  - Base: Negro Profundo (#000000)
  - Acento: `forge-ember` (#FF3B2F) INCANDESCENTE.
  - Acabado: Vidrio volcánico + Luz propia (Glow).
